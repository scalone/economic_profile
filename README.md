# Economic Profile
## Overview
Academic project to generate graphs from a research conducted with Fatec students.

## Installation
Execute:

	$ git clone git@github.com:scalone/economic_profile.git
	$ cd economic_profile && bundle install

## Execution
Execute:

	$ rackup

## Usage
Open on Browser:

	http://localhost:9292

## Tests
##### Unit

	$ rake test

##### Guard

	$ guard start
